My Projects

Exam-Wise Summarisation Tool

Developed a web application that assists students in preparing for exams by providing concise summaries of relevant topics. The tool leverages web scraping to extract educational content from websites like Javatpoint. Using SpaCy in Python, the extracted text is processed and summarized into manageable, exam-focused information. The application is built with Flask for seamless deployment, ensuring an intuitive and accessible user interface for end-users.

E-Commerce Website

Developed a full-stack e-commerce website that enables both buyers and sellers to interact with ease. The platform provides CRUD operations for sellers to manage product listings, while buyers can browse, add to cart, and place orders. The website is built using HTML, CSS, and JavaScript for the front end, with Java Servlets handling the backend logic for seamless user interactions. Data is managed through JDBC to communicate with a MySQL database, ensuring robust performance and smooth operation across the application.
